# 🌊 台灣海平面上升與地勢淹水模擬互動地圖 (Taiwan Flood Simulator)

這是一個兼具**科學精準度（Scientific Accuracy）**與**極致視覺美學（Rich Aesthetics）**的專業級網頁版台灣淹水模擬工具。結合了高解析度地形起伏模型（DEM）、WebGIS 地圖框架（Leaflet）、即時水文物理連通性演算法（Optimized BFS）以及豐富的互動指標。

系統旨在提供學術研究、防災教育以及政策評估一項直觀、精準且流暢的極端水位上升模擬工具。

---

## ✨ 核心特色與技術升級

### 1. 水文連通性模擬 (BFS Flood Fill)
相較於簡單的「數值高低篩選」導致內陸盆地亂淹的邏輯錯誤，本系統採用了**廣度優先搜尋 (BFS)** 演算法。只有與海岸線（海洋）有物理連通性的低窪地區才會被洪水填充，精準模擬海水倒灌與溢流的真實物理路徑。

### 2. 科學潮位與氣候暴潮疊加 (Tide & Surge Coupling)
模擬水位不再只是單一的靜態參數。系統基於科學防災公式：
$$\text{Total Water Level} = \text{Mean Sea Level (MSL)} + \text{Tide Offset} + \text{Surge Offset}$$
在海平面上升控制項下設計了動態「算式看板」，當使用者切換日常高潮 (+1.5m) 或颱風暴潮 (+3.5m) 等氣候情境時，看板會即時列出物理疊加算式（例如：`實質水位: MSL (5.0m) + 氣候差值 (+1.5m) = 6.5m`），清晰展現環境變數的疊加效應。

### 3. 雙軌地形高程解碼 (Dual-Decoding Modes)
支援在「高程數據校正」卡片中即時切換解碼模式：
*   **直譯模式 (Legacy Mode)**：$1\text{ unit} = 1\text{ meter}$，垂直解析度為 **1 公尺**（解碼公式：$R \times 256 + G$）。
*   **科學歸一化模式 (Scientific Mode)**：將實質海拔 $[0, 4000\text{m}]$ 線性映射至 16-bit 的整數空間 $[0, 65535]$，垂直解析度精細提升至 **$6.1\text{公分}$**！大幅消除了等高線邊界的鋸齒狀噪點，呈現完美的地理等高邊界。

### 4. 雙層防禦阻隔控制 (Dual-Layer Constraints)
*   **靜態約束遮罩層 (Impenetrable Mask)**：支援非阻塞式加載地理障礙遮罩圖層 `assets/taiwan_mask.png`。遮罩區域在模擬中直接作為**無限高/不可穿透的物理屏障**，阻斷海水倒灌。
*   **動態人工防洪堤 (Overtoppable Levees)**：使用者可在地圖上即時點擊**繪製自定義防洪堤防線**，並透過滑桿動態調整堤高（如 6m）。水位低於 6m 時成功阻水；當水位超過 6m 時，海水會真實地**越頂溢流 (Over-topping)** 過去，完美契合真實潰堤水文情況！

### 5. 極致的毫秒級運算效能 (Optimized BFS)
為解決前端 JavaScript 在高頻滑動時可能產生的卡頓，進行了極致的效能重構：
*   **海洋像素預快取 (Ocean Pre-caching)**：初始化時將所有海洋種子像素快取在一維 `Int32Array` 中，種子注入複雜度直接降為 $O(1)$。
*   **一維坐標偏移搜尋 (1D Offset Search)**：BFS 迴圈中徹底移除了取整、除法與鄰居迴圈，改為直接對一維陣列索引進行上下左右邊界判定與偏移計算，運算效率提升 **3x 至 5x**。

### 6. 衝擊與受災指標即時評估
*   **區域淹水狀態**：台北盆地、宜蘭平原、嘉南平原、高雄沿海及花東海岸等 5 大政經精華區的淹水比例與警戒狀態監控。
*   **受影響人口動態估算**：基於台灣人口高程分佈模型，利用連通性淹水比例即時計算並更新估算受災人口數。
*   **全球衝擊側邊欄**：包含中國主要省份與世界各大指標沿海城市（如阿姆斯特丹、威尼斯、新加坡）在同等水位下的受災狀態聯動。
*   **滑鼠位置即時查詢**：滑鼠移至地圖任意點即可顯示精確經緯度、海拔高度、以及狀態評估（安全/已淹沒/堤防保護區）。

## 📂 專案結構

*   **`index.html`**：前端網頁結構，採用磨砂玻璃（Glassmorphism）與高質感深色系主題。
*   **`app.js`**：核心模擬引擎、Leaflet 地圖控制、BFS 演算法與繪圖邏輯。
*   **`styles.css`**：全局樣式設計系統，包含客製化地圖元件、動畫效果及響應式布局。
*   **`assets/taiwan_data.js`**：核心地形數據檔，以 Base64 DataURL 嵌入 16-bit 地形高程圖及地理邊界資訊。
*   **`assets/taiwan_mask.png`** *(選用)*：不可穿透的靜態防禦/地景約束遮罩圖層。
*   **`tools/convert_dem_to_png.py`**：高精度離線地形數據轉換 Python 腳本。

## 🛠️ 地形數據轉換指引 (convert_dem_to_png.py)

如果您想要導入政府地理資訊平台（如 TGOS）下載的真實高解析度高程數據，請使用專屬腳本進行轉換。

### 依賴安裝
轉換腳本需要 `Pillow` 與 `numpy`。如果您有專業的地理空間網格檔案（GeoTIFF），建議安裝 `rasterio` 以保留完整空間坐標資訊：
```bash
pip install Pillow numpy rasterio
```

### 轉換指令範例

#### 1. 科學歸一化模式 (推薦，6.1cm 垂直精度)
將 GeoTIFF 檔案編碼為 16-bit 線性對應的高精度 PNG：
```bash
python tools/convert_dem_to_png.py -i input_dem.tif -o assets/taiwan_elevation.png --mode normalized --min 0 --max 4000
```

#### 2. 直譯模式 (1m 精度)
將 ASCII 網格檔編碼為 $1\text{ unit} = 1\text{ meter}$ 的直譯式 PNG：
```bash
python tools/convert_dem_to_png.py -i input_dem.asc -o assets/taiwan_elevation.png --mode direct
```

### PNG 頻道編碼結構
轉換後的 PNG 圖片包含三個頻道的物理意義：
*   **Red (R)**：高程整數的高 8 位。
*   **Green (G)**：高程整數的低 8 位。
*   **Blue (B)**：陸海遮罩（`255` 代表陸地，`0` 代表海洋/無數據）。

## 🚀 快速開始

本專案無需繁瑣的後端架構，完全在瀏覽器客戶端進行高效率運算。

### 本地運行
1.  複製（Clone）本專案至本地。
2.  雙擊 **`index.html`** 或使用 VS Code 的 **Live Server** 套件開啟。
3.  調整左側控制面板的「海平面上升控制」滑桿，即可即時看到淹水擴散渲染效果！

### 繪製堤防教學
1.  在左側側邊欄找到「防禦工程與屏障」卡片。
2.  將「啟用人工防禦堤防」開關打開，此時會展開防線繪製控制項。
3.  調整「設定堤防設計高度」滑桿（例如設定為 $6\text{m}$）。
4.  點擊「繪製防洪堤」，此時地圖鼠標將變為十字準心。
5.  在 Leaflet 地圖上點擊滑鼠左鍵，劃定堤防線段。
6.  **雙擊滑鼠左鍵**或**按鍵盤 Esc 鍵**即可完成繪製。
7.  拉動海平面滑桿至 $5\text{m}$，您會發現海水成功被阻隔在堤防外；拉升至 $7\text{m}$ 時，海水將會溢流翻越該堤防！
